# Find Factors

## Feature

1. Create a function that factors an integer into primes
2. The function argument is an integer
3. The function returns a dictionary; the key is a factor, and the value is the exponent for the factor.

## Style

1. Function names are camel case.
2. Function docstrings include: function signature, description of its operation, a usage example
3. Variable names.
   1. Only lists end in "s"
   2. Arrays end in "_arr"
   3. Variables are lower case with underscore ("_")

## Tests

1. Use the the following guard for all tests
        If IGNORE_TESTS:
            return
2. Initialize IGNORE_TESTS to ``False``
